Retrieve a map of attribute values at a constant specified depth on a regular latitude-longitude grid.

For every point on the map, if the specified depth is greater than the depth of the bottom of the specified layer, then attribute values are interpolated at the bottom of the specified layer. Similarly, if the specified depth is less than the depth of the top of the specified layer, then attribute values at the top of the specified layer are interpolated. It is the responsibility of the user to deallocate the returned array.

Parameters
modelthe GeoTessModel to be interrogated.
latitudesarray of latitude values in degrees.
numLatnumber of latitudes
longitudesarray of longitude values in degrees.
numLonnumber of longitudes
layerIdlayer index
depththe depth at which samples should be interpolated, in km.
horizontalTypeeither InterpolatorType.LINEAR or InterpolatorType.NATURAL_NEIGHBOR
radialTypeeither InterpolatorType.LINEAR or InterpolatorType.CUBIC_SPLINE
reciprocalif false, return value; if true, return 1./value.
attributesindex(es) of the attributes to interpolate.
numAttrnumber of attributes
values(output) 3D vector of doubles: [nlat][nlon][nAttributes]
nx(output) number of latitudes
ny(output) number of longitudes
nz(output) number of attributes

Retrieve a map of attribute values at some fractional thickness with a layer.

FractionalRadius <= 0. will result in samples at the bottom of the layer while fractionalRadius >= 1. will result in samples at the top of the layer. It is the responsibility of the user to deallocate the returned array.

Evaluates maximum number of nodes per layer.

On input, pointsPerLayer is an array of length nLayers where each element contains a current estimate of the number of nodes that must be deployed on the corresponding layer so that the node spacing will be no greater than maxSpacing (in km). The values in pointsPerLayer will be evaluated at the specified position and increased if necessary. Only layers between firstLayer and lastLayer, inclusive, will be evaluated.
