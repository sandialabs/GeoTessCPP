enum InterpolatorTypeC

An enum list of all the different types of interpolation that GeoTess knows how to perform.

Includes both 2D interpolation algorithms that will be applied to vertices in the 2D grid, and 1D interpolation algorithms that will be applied to nodes distributed along radial profiles.
